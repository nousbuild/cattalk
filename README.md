### 代码信息

#### CatTalk 漫画网站

小喵心语原创校园漫画《那些年的校园与我们》官网网站。

#### 开发语言

HTML, JavaScript, CSS

#### 服务器环境

仅需储存空间即可

#### 依赖

依赖于 Live2D API 接口

- `Live2D API` https://www.nousbuild.org/api/live2d/

#### 如何使用 ?

1. 获取某个分支的代码文件到本地，不保留根文件夹 `cattalk.nousbuild.org`；
2. 上传所有文件至服务器根目录 `htdocs` 或上传至 OSS；
